Instruction Set

Opcode Mnemonic Description
0001LDA addrLoad value from memory address into A register.
0010ADD addrAdd value at memory address to A register.
0011SUB addrSubtract value at memory address from A register.
0100STA addrStore A register value into memory address.
0101LDI valueLoad immediate 4-bit value directly into A register.
0110JMP addrJump to memory address unconditionally.
0111JPC addrJump if Carry flag is set.
1000JPZ addrJump if Zero flag is set.
1110OUTSend A register value to output display.
1111HLTHalt the computer. Stop execution.
